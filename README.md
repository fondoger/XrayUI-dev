<h1 align="center">XrayUI</h1>
A native Windows GUI client for the Xray core, built with <a style="text-decoration:none" href="https://docs.microsoft.com/windows/apps/winui">WinUI</a>. Designed to be a fast, lightweight, and full-featured proxy client.


## Features

- Support Shadowsocks, VMess, VLESS, Trojan, Hysteria2 and Chain Proxy
- TUN mode
- Subscription import and update
- AI Unlock Status Detection
- Custom routing rules with geoip / geosite
- Auto-start on boot, auto-connect
- Theme and protocol color customization

## Build

Requires .NET 10 SDK and Windows 10 1809 or later.

    dotnet build -c Release
    dotnet publish -c Release -r win-x64
